# Jacob Campbell - Personal Portfolio Website

A clean, modern, and responsive personal portfolio website showcasing Jacob Campbell's work as a Software Engineer. This site includes three main sections: About/Contact, Projects, and Resume/Links, with a professional design optimized for GitHub Pages deployment and NFC business card integration.

## 🌟 Features

- **Responsive Design**: Looks great on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean design with smooth animations and hover effects
- **Three Main Sections**:
  - About/Contact with downloadable vCard (JacobCampbell.vcf)
  - Projects showcase featuring Jacob's 3 key projects
  - Resume/Links section with social media integration
- **Performance Optimized**: Fast loading with lazy loading images
- **Accessibility**: Screen reader friendly with proper ARIA labels
- **SEO Ready**: Proper meta tags and semantic HTML structure
- **Easy to Customize**: Simple HTML/CSS/JS structure for easy modifications

## 🚀 Quick Start & Deployment

### GitHub Pages (Recommended)
This repository is ready for GitHub Pages deployment:

1. **Automatic Deployment**: The site is configured to deploy automatically when changes are pushed to the main branch
2. **Custom Domain**: Can be configured in repository settings if desired
3. **Access**: Once deployed, the site will be available at `https://soupyofficial.github.io`

### Alternative Deployment Options

#### Vercel
1. Connect this repository to [Vercel](https://vercel.com)
2. Deploy with default settings
3. Enjoy fast global CDN and automatic deployments

#### Netlify
1. Connect this repository to [Netlify](https://netlify.com)
2. Deploy with default build settings
3. Includes form handling and custom domains

### Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/SoupyOfficial/SoupyOfficial.github.io.git
   cd SoupyOfficial.github.io
   ```
2. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (with http-server)
   npx http-server
   ```
3. Visit `http://localhost:8000` in your browser
